# GLD CLON'N v2.0.10

## Destaques

- Atualização profissional do empacotamento, segurança local e processo de release.

## Correções

- Melhorias de estabilidade e validação para o fluxo de build e atualização.

## Segurança e empacotamento

- DevTools e sourcemaps de produção ficam bloqueados/desativados.
- Backend local usa autenticação por sessão no app empacotado.
- Backend é preparado para distribuição bundlada/minificada.
- Artefatos de release recebem checksums SHA-256 públicos.
